## 🚀 Features

- 🔍 Full website crawling for internal links
- 🧠 Smart form detection using BeautifulSoup
- 🧪 Automated injection of XSS & SQLi payloads
- 📋 Real-time vulnerability detection & result display
- 🖥️ Interactive Flask-based web interface
- 📄 Markdown report output for documentation

---

## 🧰 Tech Stack

| Tool            | Description                           |
|------------------|---------------------------------------|
| 🐍 Python 3.10+    | Programming language                 |
| 🔎 Requests       | HTTP requests to fetch pages         |
| 🍲 BeautifulSoup  | HTML parsing and form scraping       |
| 🌐 Flask          | Web framework for user interface     |
| 📝 Markdown       | For output report `report.md`        |
